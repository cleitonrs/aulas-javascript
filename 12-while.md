## while loop

~~~
let i = 0

while (i < 5) {
  console.log(`Dentro do loop: ${i}`)
  i++ 
}
~~~

Saída do **console.log()** -> 

Dentro do loop: 0

Dentro do loop: 1

Dentro do loop: 2

Dentro do loop: 3

Dentro do loop: 4

##

~~~

const names = ['Jobs', 'Hopper', 'Turing']

let i = 0

while (i < names.length) {
  console.log(names[i])
  i++
}
~~~
Saída do **console.log()** -> 

Jobs

Hopper

Turing
